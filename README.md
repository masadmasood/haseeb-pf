# Asad Masood Portfolio

Modern portfolio built with Next.js 16, React 19, TypeScript, and Tailwind CSS v4.

## Stack

- Next.js App Router
- React 19
- TypeScript
- Tailwind CSS v4
- GSAP
- Lucide React

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

## Notes

- Icons are standardized on `lucide-react`.
- Shared site metadata lives in `src/lib/site.ts`.
- Case study content is managed through `src/lib/case-studies.ts`.
